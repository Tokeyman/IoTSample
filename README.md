# IoTSample
>在UWP平台上使用的常用类库，主要包含以下类库 
> * GLibrary.Device.GPIO
> * GLibrary.Device.Net
> * GLibrary.Device.UART

## GLibrary.Device.GPIO ##
通用IO操作类，主要操作Windows 10 IoT Core上的GPIO操作，做了一点点封装，省略了自带库的初始化等操作。
## GLibrary.Device.Net ##
包含常用的UDP客户端(UdpClient)，UDP服务端(UdpServer),TCP客户端(TcpClient)，TCP服务端(TcpServer)
## GLibrary.Device.UART ##
串口操作封装
